```
├── 比較記事 → 対象車種の記事が先に必要

├── 全エリア記事 → エリアデータの整備
├── GEO最適化 → コンテンツ量が前提

Phase 4（Phase 3完了後）
├── データレポート → 十分なデータ蓄積
└── API提供 → 技術基盤の安定
```
