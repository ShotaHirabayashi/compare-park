## コンテンツ制作ルール

### 品質基準
- 1記事あたり1,500-3,000文字（車種記事）/ 2,000-5,000文字（知識記事）
- 必ず判定テーブル（寸法×制限の比較表）を含む
- トメピタの判定ツールへの内部リンクを含む
- updatedAtを設定し、データ変更時にリライト
- 「立体駐車場」を同義語として含める

### SEOチェックリスト（記事公開前）
- [ ] titleは50-60文字、主要キーワードを前方に配置
- [ ] descriptionは120文字以内
- [ ] h2から開始、h1は使わない
- [ ] carSlugを設定（車種記事の場合）
- [ ] テーブルで寸法データを明示
- [ ] 判定ツールへの内部リンクあり
- [ ] 関連車種・関連記事への内部リンクあり
- [ ] 更新日を設定
