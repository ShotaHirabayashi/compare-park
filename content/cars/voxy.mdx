## ヴォクシーの寸法スペック

トヨタ ヴォクシーは、ファミリー層に人気のミドルサイズミニバンです。アルファードより一回り小さいものの、全高は1,895mmと高く、機械式駐車場では注意が必要です。

### 90系（現行モデル）の寸法

| 項目 | 標準ボディ | エアロボディ |
|------|-----------|-------------|
| 全長 | 4,695mm | 4,695mm |
| 全幅 | 1,730mm | 1,730mm |
| 全高 | 1,895mm | 1,895mm |
| ホイールベース | 2,850mm | 2,850mm |

### 80系（先代モデル）の寸法

| 項目 | 標準ボディ | エアロボディ |
|------|-----------|-------------|
| 全長 | 4,695mm | 4,710mm |
| 全幅 | 1,695mm | 1,735mm |
| 全高 | 1,825mm | 1,825mm |
| ホイールベース | 2,850mm | 2,850mm |

## ノアとの違い

トヨタ ノアはヴォクシーの兄弟車で、基本的なボディサイズは同一です。デザインの違い（フロントグリルなど）により全長が数mm異なる場合がありますが、駐車場適合の判定では**ほぼ同じ**と考えて問題ありません。

## 機械式駐車場に入るか？

**ヴォクシーは一般的な機械式駐車場（全高1,550mm制限）には入りません。ハイルーフ対応の駐車場でも全高1,895mmのため、入れない場合が多いです。**

### 各サイズ制限での判定

| 制限 | 全長 | 全幅 | 全高 | 判定 |
|------|------|------|------|------|
| 普通車（1550mm） | OK | OK | **NG** | ❌ NG |
| ハイルーフ（1800mm） | OK | OK | **NG** | ❌ NG |
| ハイルーフ（2000mm） | OK | OK | OK | ✅ OK |
| 大型車（2100mm） | OK | OK | OK | ✅ OK |

### 注意点

ヴォクシーの全高1,895mmは、ハイルーフ対応の中でも**制限値が1,900mm以上**の駐車場を選ぶ必要があります。ハイルーフ対応と表記されていても、全高制限が1,800mmの駐車場では入りません。

## ヴォクシーが入る駐車場の条件

- **全高制限**: 1,900mm以上（余裕を持って2,000mm以上が望ましい）
- **全長制限**: 4,700mm以上
- **全幅制限**: 1,750mm以上
- **重量制限**: 1,700kg以上

## ヴォクシーオーナーの駐車場事情

ファミリー層がメインのヴォクシーは、マンションの駐車場利用が多い車種です。しかし、マンションの機械式駐車場は普通車規格が多く、以下のような問題に直面することがあります。

- マンション購入後に機械式駐車場に入らないと分かった
- 空いているハイルーフ区画がない
- 近隣の月極駐車場を借りることになった

**マンション購入・引っ越し前に、駐車場のサイズ制限を必ず確認しましょう。**
