## N-BOXの寸法スペック

ホンダ N-BOXは、日本で最も売れている軽自動車です。広い室内空間を実現するため全高が高いスーパーハイトワゴンタイプで、機械式駐車場では全高制限に注意が必要です。

### 3代目 JF5/JF6（現行・2023年〜）の寸法

| 項目 | サイズ |
|------|--------|
| 全長 | 3,395mm |
| 全幅 | 1,475mm |
| 全高 | 1,790mm（2WD）/ 1,815mm（4WD） |
| 重量 | 910〜990kg |

### 2代目 JF3/JF4（先代・2017-2023年）の寸法

| 項目 | サイズ |
|------|--------|
| 全長 | 3,395mm |
| 全幅 | 1,475mm |
| 全高 | 1,790mm（2WD）/ 1,815mm（4WD） |
| 重量 | 890〜990kg |

## 高さ制限との比較 — 機械式駐車場に入るか？

**N-BOXは全長・全幅は余裕で収まりますが、全高がネックです。**

軽自動車なので全長3,395mm・全幅1,475mmはどの機械式駐車場でも問題ありません。しかし全高1,790〜1,815mmは、一般的な普通車サイズの機械式駐車場（全高1,550mm制限）には入りません。

### 各高さ制限・サイズ制限での判定

| 制限 | 全長 | 全幅 | 全高 | 重量 | 判定 |
|------|------|------|------|------|------|
| 普通車（1550mm） | OK | OK | **NG** | OK | ❌ NG |
| ハイルーフ（1800mm） | OK | OK | OK（2WD）/ ⚠️（4WD） | OK | ⚠️ 要確認 |
| 大型車（2000mm） | OK | OK | OK | OK | ✅ OK |

### 2WDと4WDで全高が異なる

N-BOXは**2WDと4WDで全高が25mm異なる**点に注意が必要です。

- **2WD**: 全高1,790mm → ハイルーフ対応（1,800mm）に**余裕10mmで収まる**
- **4WD**: 全高1,815mm → ハイルーフ対応（1,800mm）を**15mm超過でNG**

4WDモデルの場合、全高1,800mm制限の駐車場は利用できません。

## N-BOXが入る駐車場の条件

### 2WDモデル
- **全高制限**: 1,800mm以上であればOK
- 全長・全幅・重量はほぼ全ての機械式駐車場で問題なし

### 4WDモデル
- **全高制限**: 1,850mm以上が安心
- 1,800mm制限では15mm超過でNG
