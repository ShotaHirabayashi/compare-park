## アルファードの寸法スペック

トヨタ アルファードは、国産ミニバンの中でも最大級のボディサイズを誇ります。駐車場選びで最も注意が必要な車種の一つです。

### 40系（現行モデル）の寸法

| 項目 | サイズ |
|------|--------|
| 全長 | 4,995mm |
| 全幅 | 1,850mm |
| 全高 | 1,935mm |
| ホイールベース | 3,000mm |

### 30系（先代モデル）の寸法

| 項目 | サイズ |
|------|--------|
| 全長 | 4,945〜4,950mm |
| 全幅 | 1,850mm |
| 全高 | 1,935〜1,950mm |
| ホイールベース | 3,000mm |

## 高さ制限との比較 — 機械式駐車場に入るか？

結論から言うと、**アルファードは一般的な機械式駐車場にはほぼ入りません**。

### 入らない理由

機械式駐車場のサイズ制限は主に3つの規格に分かれます。

1. **普通車サイズ**：全高1,550mm以下 → **NG**（アルファードは1,935mm）
2. **ハイルーフ対応**：全高1,800〜2,000mm → **車高はギリギリだが、全長・全幅で制限超えの場合あり**
3. **大型車対応**：全長5,300mm・全幅2,050mm・全高2,000mm以下 → **OK**

### 各高さ制限・サイズ制限での判定

| 制限 | 全長 | 全幅 | 全高 | 判定 |
|------|------|------|------|------|
| 普通車（1550mm） | OK | OK | **NG** | ❌ NG |
| ハイルーフ（1800mm） | ギリギリ | OK | ギリギリ | ⚠️ 要確認 |
| 大型車（2000mm） | OK | OK | OK | ✅ OK |

## アルファードが入る駐車場の条件

アルファードを停められる機械式駐車場を探す場合、以下の条件を満たす必要があります。

- **全高制限**: 1,950mm以上（40系の場合1,935mm＋余裕）
- **全長制限**: 5,000mm以上
- **全幅制限**: 1,850mm以上（ミラー折りたたみ時）
- **重量制限**: 2,200kg以上

## タワー式駐車場の場合

タワー式（エレベーター式）駐車場は、機械式の中でも比較的大きなサイズに対応しているものがあります。ただし、施設ごとにサイズ制限が異なるため、事前確認は必須です。
