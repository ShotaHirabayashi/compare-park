## フリードの寸法スペック

ホンダ フリードは、5ナンバーサイズのコンパクトミニバンとして高い人気を誇ります。コンパクトなボディに3列シートを備えた実用的な設計ですが、全高が高いため機械式駐車場では注意が必要です。

### 3代目（現行・2024年〜）の寸法

| 項目 | サイズ |
|------|--------|
| 全長 | 4,310mm |
| 全幅 | 1,695mm |
| 全高 | 1,755mm（2WD）/ 1,780mm（4WD） |
| 重量 | 1,370〜1,450kg |

### 2代目 GB5/6/7/8（先代・2016-2024年）の寸法

| 項目 | サイズ |
|------|--------|
| 全長 | 4,265mm |
| 全幅 | 1,695mm |
| 全高 | 1,710〜1,735mm |
| 重量 | 1,340〜1,490kg |

## 機械式駐車場に入るか？

**フリードはハイルーフ対応の機械式駐車場であれば、ほぼ問題なく入ります。**

全長・全幅は5ナンバーサイズで余裕があり、重量も軽量なため制限に引っかかることはまれです。唯一の注意点は**全高**で、普通車サイズの機械式駐車場には入りません。

### 各サイズ制限での判定（3代目・2WD）

| 制限 | 全長 | 全幅 | 全高 | 重量 | 判定 |
|------|------|------|------|------|------|
| 普通車（1550mm） | OK | OK | **NG** | OK | ❌ NG |
| ハイルーフ（1800mm） | OK | OK | OK | OK | ✅ OK |
| 大型車（2000mm） | OK | OK | OK | OK | ✅ OK |

### 2WDと4WDの違い

フリード3代目は、2WDと4WDで全高が25mm異なります。

- **2WD**: 全高1,755mm → ハイルーフ対応（1,800mm）に**余裕45mmでOK**
- **4WD**: 全高1,780mm → ハイルーフ対応（1,800mm）に**余裕20mmでOK**

どちらもハイルーフ対応であれば問題なく入ります。

## フリードが入る駐車場の条件

- **全高制限**: 1,800mm以上（2WD/4WDとも余裕あり）
- **全長制限**: 4,400mm以上（ほぼ全ての駐車場でクリア）
- **全幅制限**: 1,700mm以上（5ナンバーなので余裕）
- **重量制限**: 1,500kg以上（ほぼ全ての駐車場でクリア）

## 同クラスのコンパクトミニバンとの比較

| 車種 | 全長 | 全幅 | 全高（2WD） | ハイルーフ1800mm |
|------|------|------|-----------|----------------|
| フリード（3代目） | 4,310mm | 1,695mm | 1,755mm | ✅ OK |
| シエンタ（3代目） | 4,260mm | 1,695mm | 1,695mm | ✅ OK |
| ルーミー | 3,700mm | 1,670mm | 1,735mm | ✅ OK |

フリードはコンパクトミニバンの中でも全高が高めですが、ハイルーフ対応であれば問題ありません。ライバルのシエンタは全高1,695mmとフリードより60mm低く、よりスムーズに入ります。

## まとめ

- フリードは**全高1,755〜1,780mm**のため、普通車サイズの機械式駐車場（全高1,550mm制限）には入りません
- **ハイルーフ対応（全高1,800mm以上）** の駐車場であれば、2WD・4WDとも問題なく入ります
- 全長・全幅は5ナンバーサイズで余裕があり、重量も軽量です
- コンパクトミニバンの中では駐車場の選択肢が比較的広い車種です
- 駐車場ごとに制限が異なるため、トメピタで事前に確認することをおすすめします
