**【結論】シエンタはハイルーフ対応の機械式駐車場に余裕を持って入る。** 全高1,695mm（2WD）はハイルーフ制限（1,800mm）に対して105mmの余裕があり、5ナンバーサイズの全幅1,695mm・全長4,260mmも全く問題なし。ミニバンの中で機械式駐車場に最も入りやすい車種です。

## シエンタの寸法スペック

トヨタ シエンタは、5ナンバーサイズのコンパクトミニバンです。フリードと並ぶ人気車種で、全高が低めに設計されているため、ミニバンの中では**機械式駐車場に最も入りやすい車種**の一つです。

### 3代目（現行・2022年〜）の寸法

| 項目 | サイズ |
|------|--------|
| 全長 | 4,260mm |
| 全幅 | 1,695mm |
| 全高 | 1,695mm（2WD）/ 1,715mm（4WD） |
| 重量 | 1,270〜1,410kg |

## 高さ制限との比較 — 機械式駐車場に入るか？

**シエンタはハイルーフ対応の機械式駐車場であれば余裕を持って入ります。**

5ナンバーサイズで全長4,260mm・全幅1,695mmは全く問題なし。全高1,695mmはハイルーフ対応（1,800mm）に対して**105mmの余裕**があり、ミニバンの中で最も機械式駐車場に入りやすいです。

### 各高さ制限・サイズ制限での判定

| 制限 | 全長 | 全幅 | 全高 | 重量 | 判定 |
|------|------|------|------|------|------|
| 普通車（1550mm） | OK | OK | **NG** | OK | ❌ NG |
| ハイルーフ（1800mm） | OK | OK | OK | OK | ✅ OK |
| 大型車（2000mm） | OK | OK | OK | OK | ✅ OK |

### 2WDと4WDの違い

| 駆動方式 | 全高 | 1,800mm制限 | 余裕 |
|---------|------|-----------|------|
| 2WD | 1,695mm | ✅ OK | 105mm |
| 4WD（E-Four） | 1,715mm | ✅ OK | 85mm |

どちらも余裕を持ってハイルーフ対応に収まります。

## ライバル車との比較

| 車種 | 全高（2WD） | 1,800mm制限 | 余裕 |
|------|-----------|-----------|------|
| **シエンタ** | **1,695mm** | **✅ OK** | **105mm** |
| フリード | 1,755mm | ✅ OK | 45mm |
| ルーミー | 1,735mm | ✅ OK | 65mm |
| ソリオ | 1,745mm | ✅ OK | 55mm |

シエンタはコンパクトミニバン・ハイトワゴンの中で**最も全高が低く**、機械式駐車場に最も入りやすい車種です。

## シエンタが入る駐車場の条件

- **全高制限**: 1,700mm以上あれば安心（2WD）/ 1,750mm以上（4WD）
- **全長制限**: 4,300mm以上（ほぼ全ての駐車場でクリア）
- **全幅制限**: 1,700mm以上（5ナンバーなので余裕）
- **重量制限**: 1,500kg以上（余裕あり）
